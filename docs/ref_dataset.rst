Module **dataset**
==================

The class :class:`MoccaDataset<mocca2.dataset.dataset.MoccaDataset>` provides high-level interface for managing datasets of chromatograms and libraries of compounds.

ProcessingSettings
~~~~~~~~~~~~~~~~~~
The ProcessingSettings class contains all settings for automatic batch processing of batches of chromatograms. It is primarily used by the :class:`MoccaDataset<mocca2.dataset.dataset.MoccaDataset>` class.
